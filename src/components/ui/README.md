# `components/ui/`

Primitives — generic, app-agnostic UI components.

- Buttons, inputs, dialogs, cards, badges, tooltips, etc.
- Source: shadcn/ui (copy-paste, owned in our repo — not a dependency).
- **No business logic.** Receive props, render. Composable.

Naming: `kebab-case.tsx` (e.g. `button.tsx`, `dropdown-menu.tsx`).
